# Gyroscope-Controlled-Car

A car made up of 3 BO motors (2 Single Shaft and 1 Dual Shaft), 60mm tyres, L298N Motor Driver, Arduino Uno R4 Wifi and HC05 Bluetooth module, is controlled wirelessly over Bluetooth by a controller attached on a glove, worn by the User. 

The Controller consist of Arduino Nano, MPU6050 Gyroscope sensor and HC05 Bluetooth module.

The Controller reads the Pitch and Roll of the hand of the user and transmit this information to the car, where this information is read and the corresponding speed and direction of the motor is decided.
